# Python Password Strength Analyzer

A program that evaluates password strength and identifies common password security weaknesses.

</div>

---

## Description

This is a simple password analyzer that will try to find weaknesses in the user's password,
and grade them from 0-100 based on how strong they are.

---

## Features

<ul>
  <li>Evaluates password length</li>
  <li>Checks for uppercase, lowercase, numeric, and special characters</li>
  <li>Detects common passwords</li>
  <li>Detects repeated character patterns</li>
  <li>Checks whether the password contains the user's name</li>
  <li>Calculates a password strength score out of 100</li>
  <li>Classifies passwords from Very Weak to Very Strong</li>
</ul>
